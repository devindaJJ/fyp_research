1. SYSTEM OVERVIEW - WHAT'S HAPPENING
================================================================================

This is an INTELLIGENT TRAFFIC MONITORING SYSTEM that combines multiple AI/ML 
technologies to create a comprehensive, real-time traffic safety and enforcement 
platform.

WHAT THE SYSTEM DOES (Frame-by-Frame):
---------------------------------------
1. VEHICLE DETECTION & TRACKING
   - Detects vehicles using YOLOv8 (car, motorcycle, bus, truck)
   - Tracks each vehicle across frames with unique ID (DeepSORT)
   - Records vehicle position (bounding box) at every frame

2. SPEED CALCULATION
   - Measures speed using virtual timing lines
   - Calculates: speed = distance / time
   - Detects speeding violations automatically

3. NUMBER PLATE RECOGNITION (ANPR)
   - Detects license plates using YOLO
   - Reads plate text using OCR (EasyOCR)
   - Validates Sri Lankan plate formats

4. INTELLIGENT LINKING
   - Matches detected plates to tracked vehicles
   - Uses bounding box overlap (IoU) algorithm
   - Creates unified record per vehicle: {ID, Speed, Plate, Violations}

5. GOOGLE MAPS INTEGRATION (Optional)
   - Fetches real speed limits from Google Maps API
   - Detects special zones (schools, hospitals)
   - Adjusts enforcement rules dynamically

6. REAL-TIME SAFETY DETECTION ⭐ NOVEL FEATURE
   - Detects dangerous driving behaviors:
     * Aggressive acceleration (>10 km/h per second)
     * Hard braking (>8 km/h per second)
     * Erratic movement/lane weaving (>40px lateral shift)
   - Detects vehicle interactions:
     * Collisions (>50% bounding box overlap)
     * Near-misses (<100px distance + high relative speed)
   - Generates instant alerts with timestamp

7. DATA EXPORT & REPORTING
   - Exports to CSV, JSON formats
   - Generates violation-only reports
   - Creates comprehensive statistics

================================================================================

MODULE STRUCTURE:
-----------------

src/
├── speed/
│   └── detector.py              # VehicleSpeedDetector - Speed tracking
├── anpr/
│   ├── detector.py              # NumberPlateDetector - Plate recognition
│   ├── preprocessor.py          # Image enhancement for OCR
│   └── validator.py             # Sri Lankan plate validation
├── utils/
│   ├── integration.py           # VehicleDataIntegrator - Links everything
│   ├── exporter.py              # ResultExporter - CSV/JSON output
│   ├── google_maps_integration.py  # Dynamic speed limits
│   └── safety_detector.py       # SafetyEventDetector - NOVEL FEATURE
└── core/
    └── config.py                # Configuration management

main.py                          # Application orchestrator


WORKFLOW (Per Frame):
---------------------
Frame Input → Vehicle Detection (YOLO) → Tracking (DeepSORT) → Speed Calculation
    ↓
Update Integrator (Speed + BBox)
    ↓
ANPR Detection → Link Plates to Vehicles (IoU Matching)
    ↓
Safety Analysis (Check behaviors + interactions)
    ↓
Generate Alerts → Draw on Frame → Display → Export Results


DATA FLOW:
----------
Raw Video → Frame Processing → Vehicle Data → Integrated Records → Reports
                    ↓
              Safety Events → Real-time Alerts → Summary Statistics

7. TECHNICAL IMPLEMENTATION DETAILS
================================================================================

ALGORITHMS USED:
----------------

1. YOLOv8 (Object Detection)
   - Purpose: Detect vehicles and license plates
   - Accuracy: 85-95%
   - Speed: Real-time (30+ FPS)

2. DeepSORT (Object Tracking)
   - Purpose: Track vehicles across frames
   - Maintains unique IDs
   - Handles occlusions

3. EasyOCR (Text Recognition)
   - Purpose: Read license plate text
   - Multi-language support
   - Sri Lankan format validation

4. IoU (Intersection over Union)
   - Purpose: Match plates to vehicles
   - Geometric overlap calculation
   - Containment checking

5. Kinematic Analysis
   - Purpose: Calculate speed, acceleration
   - Physics-based calculations
   - Temporal tracking

6. Behavioral Pattern Recognition
   - Purpose: Detect dangerous behaviors
   - Threshold-based detection
   - Event correlation


PERFORMANCE OPTIMIZATION:
-------------------------
- GPU acceleration for YOLO inference
- Frame skipping for efficiency
- Result caching (Google Maps)
- Parallel processing capabilities
- Memory-efficient tracking


ACCURACY FACTORS:
-----------------
Good Performance:
✓ Clear video quality (1080p+)
✓ Good lighting conditions
✓ Visible license plates
✓ Stable camera position
✓ Proper calibration

Challenges:
✗ Low light / night footage
✗ Blurry or fast motion
✗ Occluded plates
✗ Extreme angles
✗ Weather conditions (rain, fog)
